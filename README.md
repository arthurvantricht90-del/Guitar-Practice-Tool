# Guitar-Practice-Tool
This repository contains code for a Python tool created to asssist in guitar practice.


## The Tool
This tool was created to make guitar practice easier.
The tool generates two chords at random and shows their finger positions on the fretboard.
The goal is to first recognize both chords, then the chord names can be revealed and you can check if you were right. 
A secondary goal is to then set a metronome and switch between the chords at a certain BPM to practice these praticular chord changes.

## How to run
Download the guitar_practice_tool.zip file. 
The guitar_practice_tool.zip file contains:
- main.py - this is the main code. Run from here.
- chord_library.py - this code is a library for chord configurations.

To run the code, make sure the main file and the chord library are in the same folder X. Set the working directory to that folder X and run the main script.
